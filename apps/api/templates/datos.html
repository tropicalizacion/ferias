Datos sobre las ferias

Los datos de las ferias incluyen información sobre su ubicación, horarios de apertura, servicios y otros. La información de las ferias es revisada periódicamente.

Excel CSV GeoJSON

Datos sobre los productos

Los datos de los productos incluyen información sobre cada producto vendido en las ferias del agricultor, como nombre, preparación, almacenamiento, entre otros.

Excel CSV JSON

API deferia.cr

deferia.cr ofrece un RESTful API para acceder a los datos de las ferias.

Raíz del API del REST Framework de Django

Documentación completa del API

Ejemplos
GET deferia.cr/datos/api/ferias
Lista de todas las ferias en la base de datos.
GET deferia.cr/datos/api/geoferias
Lista de todas las ferias en la base de datos en formato GeoJSON.
GET deferia.cr/datos/api/ferias/{feria_url}
Información de una feria específica.
GET deferia.cr/datos/api/productos
Lista de todos los productos y sus variedades en la base de datos.
